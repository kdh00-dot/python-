1. 숫자형 

정수 (Integer): 양의 정수, 음의 정수, 0을 포함한다

실수 (Floating-point): 소수점이 포함된 숫자

# 정수 예제
a = 10
print(a)  # 출력: 10

# 실수 예제
a = 10.5
print(a)  # 출력: 10.5

1. 8진수와 16진수

8진수 (Octal): 0o 또는 0O로 시작하며 0~7의 숫자를 사용한다

16진수 (Hexadecimal): 0x 또는 0X로 시작하며 0~9와 A~F를 사용한다

# 8진수 예제
a = 0o070
print(a)  # 출력: 56

# 16진수 예제
a = 0x1A  # 16^1*1 + 16^0*10 = 26
print(a)  # 출력: 26

2. 산술 연산자 (Arithmetic Operators)
사칙 연산
사칙 연산(+, -, *, /)을 수행

a = 3
b = 4

print(a + b)  # 출력: 7
print(a - b)  # 출력: -1
print(a * b)  # 출력: 12
print(a / b)  # 출력: 0.75

특수 연산자
제곱, 나머지, 몫을 구하는 파이썬의 유용한 연산자
제곱 연산자 (**): x의 y제곱을 구한다
나머지 연산자 (%): 나눗셈 후 나머지 값을 반환한다
몫 연산자 (//): 나눗셈 후 소수점 이하를 버리고 몫(정수값)만 반환한다

# 제곱 예제
a = 3
b = 4
print(a ** b)  # 3의 4제곱 
출력: 81

# 나머지 예제
print(7 % 3)   
# 출력: 1
print(3 % 7)   
# 출력: 3

# 몫 연산자 비교
print(7 / 4)   
# 출력: 1.75 (일반 나눗셈)

print(7 // 4)  
# 출력: 1 (몫만 반환)

3. 복합 연산자 (Compound Assignment Operators)
산술 연산자와 대입 연산자(=)를 결합하여 변수의 값을 간결하게 사용 가능하다

# 더하기 복합 연산
a = 1
a += 1  # a = a + 1과 동일
print(a)  
# 출력: 2

# 빼기 복합 연산
a = 1
a -= 1  # a = a - 1과 동일
print(a)  
# 출력: 0
